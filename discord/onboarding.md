# Discord Onboarding

## New Member Flow

1. Member joins server.
2. Admin grants `AI Access` role.
3. Member can use bot in allowed channels.
4. Bot responds only when mentioned or invoked with slash commands.

## Example Message

```text
Welcome to AI Council.

Use @Clawbot in the AI channels:
- #ask-the-oracle
- #gpt-workshop
- #claude-chamber
- #gemini-temple
- #council-debate
```
