# Permissions

Recommended Discord roles:

```text
AI Owner
AI Moderator
AI Access
```

## AI Owner

Full control.

Should usually be only Reyner.

## AI Moderator

Can help manage access and basic Discord operations.

Should not automatically get system-level access.

## AI Access

Can use the bot in allowed channels.

## Everyone

No direct access unless explicitly allowed.

## OpenClaw Policy

Prefer:

- groupPolicy: allowlist
- role-based access
- mention gating
- sandbox mode all
- workspaceAccess none or read-only
