# Council Routing

Council mode should be used when:

- user asks for comparison
- user asks for multiple opinions
- user asks "what do the AIs think?"
- decision is nuanced
- the answer benefits from trade-offs

## Council Format

```md
[Council]

## GPT View
...

## Claude View
...

## Gemini View
...

## Final Synthesis
...
```

## Rule

Do not claim that multiple models were queried unless they actually were.

If only one model is used, say:

> Simulated council-style analysis.
