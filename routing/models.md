# Model Routing

Recommended routing:

```text
#gpt-workshop      -> GPT
#claude-chamber    -> Claude
#gemini-temple     -> Gemini
#council-debate    -> Council
#ask-the-oracle    -> Router/default model
#bot-admin         -> Admin only
```

## Default Model

Use GPT as the default until Claude and Gemini are configured.

## Override Commands

Future commands:

```text
/gpt ...
/claude ...
/gemini ...
/council ...
```

## Transparency

Every response should show which model/mode is used.
