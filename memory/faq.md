# FAQ

## Why one Discord bot instead of one bot per LLM?

One bot is easier to secure, manage, log, and route.

## How do users know which model replied?

Each model-specific response should include a prefix like `[GPT]`, `[Claude]`, `[Gemini]`, or `[Council]`.

## Should everyone be owner?

No. Only Reyner should be owner by default.

## Should pairing be used?

For small private groups, role-based access is usually easier.
For public or high-risk setups, pairing is safer.
