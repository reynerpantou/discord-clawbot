# Workflows

## Add a new model

1. Add provider auth.
2. Add model prompt under `prompts/models/`.
3. Add routing rule.
4. Add Discord channel or command.
5. Test with one user.
6. Roll out to `AI Access` role.

## Add a new Discord channel

1. Create channel.
2. Ensure bot has permission.
3. Update routing docs.
4. Test mention gating.
