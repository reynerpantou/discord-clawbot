# Council Chat Example

User:

> Should I use one Discord bot for GPT/Claude/Gemini or one bot per model?

Response:

```md
[Council]

## GPT View
Use one bot and route internally. Easier ops and security.

## Claude View
Separate bots can help with identity but increase complexity.

## Gemini View
One bot with channel-based routing is more flexible for experimentation.

## Final Synthesis
Start with one bot. Add separate bots only if branding or permission boundaries require it.
```
